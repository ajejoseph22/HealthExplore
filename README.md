# fullstack-candidate-testing

## Libraries used:
* React
* React-dom
* React-modal
* Tailwind
* Next.js
* Jest

## Instruction to run the project.

* ### `npm install`
  Install dependencies.
  
* ### `npm run dev`
  Run your development server [url](http://localhost:3000).
  
* ### `npm run test`
  Run you test cases locally and on CI to ensure your application is good to deploy.
  
* ### `npm run build`
  Builts the files that are used on production.
  
* ### `npm run start`
